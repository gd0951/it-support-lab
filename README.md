# IT Support Lab – Active Directory & Jira Ticketing

## Overview
This project simulates a real-world IT support environment using Windows Server, Active Directory, and Jira Service Management.

---

## Tasks Performed
- Configured Active Directory Domain Services
- Promoted server to Domain Controller
- Created and managed user accounts
- Reset passwords and enabled/disabled users

---

## Ticketing Workflow
Used Jira Service Management to simulate help desk workflow:

### Example Ticket – Login Issue
User was unable to log in to their account.

Resolution:
- Reset password in Active Directory
- Re-enabled account
- Confirmed access restored

Status: Resolved ✅

---

## Skills Demonstrated
- Active Directory (user management)
- Password resets & account troubleshooting
- Help desk ticket workflow (Jira)
